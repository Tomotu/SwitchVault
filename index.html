Cloud Sync Setup

Sync your ideas across devices using your own free Supabase project. Data is stored privately under your passphrase.

Setup steps
1. Go to supabase.com → create a free account + new project
2. In SQL Editor, run:
create table sv_data (sync_key text primary key, data jsonb not null default '{}'::jsonb, updated_at timestamptz not null default now());
3. Go to Project Settings → API → copy Project URL + anon key
4. Paste below and choose a passphrase → Connect
5. Repeat steps 4 on your other device (same passphrase)
